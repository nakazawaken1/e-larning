# 第9回：図鑑のデータをJavaScriptで持とう！

## 今日のゴール 🎯
図鑑の情報をHTMLではなく、JavaScriptの「データ」としてまとめる！

## 学ぶこと 📚
- たくさんのデータをまとめる「配列（はいれつ）」
- 種類ごとにデータをまとめる「オブジェクト」

## やってみよう！ 💻

### ステップ1：生き物のデータをまとめよう（オブジェクト）
JavaScriptの中で、名前や写真、説明などを1つのまとまり（オブジェクト）にします。
`<script>` の中身を一度空っぽにして、新しく書き直しましょう。

```html
  <script>
    // 1つの生き物のデータ（オブジェクト）
    let asagao = {
      name: "アサガオ",
      image: "https://placehold.jp/240x180.png?text=アサガオ",
      description: "夏にきれいに咲く花です。",
      place: "学校の花壇",
      date: "7月20日"
    };

    console.log(asagao.name); // 「アサガオ」と表示されるよ！
  </script>
```

### ステップ2：リストにしてまとめよう（配列）
図鑑にはたくさんの生き物が載ります。データをリスト（配列）にしてみましょう。
`[` と `]` の間に、カンマ `,` で区切ってオブジェクトを入れていきます。

```html
  <script>
    // 生き物のデータのリスト（配列）を作る
    let zukanData = [
      {
        name: "アサガオ",
        image: "https://placehold.jp/240x180.png?text=アサガオ",
        description: "夏にきれいに咲く花です。",
        place: "学校の花壇",
        date: "7月20日"
      },
      {
        name: "カブトムシ",
        image: "https://placehold.jp/240x180.png?text=カブトムシ",
        description: "かっこいいツノがある夏の虫です。",
        place: "近くの森",
        date: "8月5日"
      }
    ];

    console.log(zukanData); // 配列の中身が見えるよ！
  </script>
```

### ステップ3：ブラウザで確認しよう
1. ブラウザを更新します。（表示されている図鑑の見た目はそのままです）
2. デベロッパーツールの「コンソール」を見てみましょう。
3. `Array(2)` のような表示が出たら、中身を開いてデータが入っているか確認してください。

## 今日のポイント 💡
- **オブジェクト** `{ name: "アサガオ", place: "花壇" }`
  - 1つのものについて、いくつかのプロパティ（名前と値）をセットにしたデータです。
- **配列（はいれつ）** `[ データ1, データ2 ]`
  - 複数のデータを順番に並べたリストです。`[]`（四角いカッコ）を使います。
- アプリ開発では、「データの管理（JS）」と「見た目の表示（HTML/CSS）」を分けて考えることがとても大切です！
