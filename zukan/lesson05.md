# 第5回：図鑑っぽくタイル状に並べよう！

## 今日のゴール 🎯
生き物のデータを横に並べて、本物の「図鑑」みたいなタイル表示にする！

## 学ぶこと 📚
- 複数の要素をグループにまとめる `<div>` タグ
- 要素を横に並べるCSS（Flexbox）

## やってみよう！ 💻

### ステップ1：グループにまとめよう（divタグ）
それぞれの生き物の情報を、`<div class="card">` という箱（タグ）で囲みます。
さらに、全部の生き物を `<div class="zukan-container">` というもっと大きな箱で囲みます。

```html
<body>
  <h1>わたしの図鑑</h1>
  
  <!-- 全体を囲む大きな箱 -->
  <div class="zukan-container">
  
    <!-- 1つ目の生き物の箱 -->
    <div class="card">
      <h2>アサガオ</h2>
      <img src="https://placehold.jp/240x180.png?text=アサガオ" alt="アサガオの写真" width="200">
      <p>夏にきれいに咲く花です。</p>
      <ul>
        <li>場所：学校の花壇</li>
        <li>日付：7月20日</li>
      </ul>
    </div>

    <!-- 2つ目の生き物の箱 -->
    <div class="card">
      <h2>カブトムシ</h2>
      <img src="https://placehold.jp/240x180.png?text=カブトムシ" alt="カブトムシの写真" width="200">
      <p>かっこいいツノがある夏の虫です。</p>
      <ul>
        <li>場所：近くの森</li>
        <li>日付：8月5日</li>
      </ul>
    </div>
    
  </div> <!-- zukan-containerの終わり -->
</body>
```

### ステップ2：CSSで横に並べよう（Flexbox）
`<style>` タグの中に、横並びにする魔法のコード `display: flex;` を追加します。

```html
<style>
  /* 前回の背景色などはそのまま残しておく */
  body {
    background-color: #f0f8ff;
  }
  h1 { color: #2e8b57; }
  h2 { color: #ff6347; }

  /* 横に並べる魔法のコード */
  .zukan-container {
    display: flex;
    flex-wrap: wrap; /* 画面に収まらない時は次の行に行く */
    gap: 20px; /* 箱と箱の間のすき間 */
  }

  /* 1つ1つの生き物の箱の設定 */
  .card {
    background-color: white; /* 箱の背景は白にする */
    width: 250px; /* 箱の横幅 */
    padding: 15px; /* 箱の内側の余白 */
  }
</style>
```

### ステップ3：ブラウザで確認しよう
ブラウザを更新して、生き物が横に並んでタイル状になったか確認しよう！

## 今日のポイント 💡
- `<div>` → 箱を作るタグです。複数のタグを1つのグループにまとめるときに使います。
- `class="名前"` → CSSでデザインを指定するために、タグに名前（クラス名）をつけます。
- `.`（ドット） → CSSでクラス名を書くときは一番前に `.` をつけます（例：`.card`）。
- `display: flex;` → 中に入っている箱を並べるための、とても便利なCSSです。
