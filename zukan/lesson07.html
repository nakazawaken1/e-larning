JavaScriptって？

今日のゴール 🎯

JavaScriptの基本を知って、コンソールにメッセージを出したり、変数を使えるようになる！

学ぶこと 📚

  • JavaScript（JS）って何のための言語？

  • デベロッパーツール（コンソール）の見方
  • データを覚えさせる「変数（へんすう）」
言語 役割
HTML 内容・構造（骨格）
CSS デザイン・見た目
JavaScript 動き・プログラム（頭脳）

やってみよう！ 💻

1<script> タグを追加しよう

</body> の直前に追加します：

  <script>
    console.log("図鑑のプログラムがスタートしました！");
  </script>
</body>

2コンソールで確認しよう

  1. ブラウザでファイルを更新する
  2. デベロッパーツールを開く（Windows: F12、Mac: Cmd+Option+I）
  3. 「Console（コンソール）」タブを開く
  4. 「図鑑のプログラムがスタートしました！」と表示されたら成功！

3変数を使ってみよう

  <script>
    // let で変数（データを入れる箱）を作る
    let aisatsu = "こんにちは！";
    console.log(aisatsu);

    let mushi = "カブトムシ";
    console.log("私が見つけたのは " + mushi + " です。");
  </script>

今日のポイント 💡

  • JavaScript（JS）はWebページに「動き」をつくるプログラミング言語
  • <script> タグの中にJSコードを書く（</body> の直前が一般的）
  • console.log("メッセージ") → 開発者向けの裏画面にメッセージを出す
  • let 名前 = "中身"; → データを覚えておく「箱（変数）」を作る
  • // から始まる行はコメント（メモ）で実行されない

自分の名前を入れる変数 let myName = "自分の名前"; を作って、コンソールに表示させてみよう！
