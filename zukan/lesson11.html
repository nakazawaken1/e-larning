第11回：図鑑に新しい
生き物を追加しよう！

今日のゴール 🎯

フォームに情報を入力して「追加」ボタンを押すと、図鑑にタイルが増えるようにする！

学ぶこと 📚

  • テキスト入力欄：<input type="text">
  • ボタン：<button>
  • 入力内容を取り出す：.value
  • 関数（ファンクション）を作る：function 名前() { }

やってみよう！ 💻

1フォームをHTMLに追加しよう

<body>
  <h1>わたしの図鑑</h1>

  <!-- 追加フォーム -->
  <div class="add-form">
    <h2>🐾 生き物を追加する</h2>
    <input id="input-name"  type="text" placeholder="名前">
    <input id="input-place" type="text" placeholder="場所">
    <input id="input-date"  type="text" placeholder="日付">
    <input id="input-memo"  type="text" placeholder="メモ">
    <button onclick="addIkimono()">追加する！</button>
  </div>

  <div id="zukan-container" class="zukan-container"></div>
</body>

3JavaScriptで追加する関数を書こう

function addIkimono() {
  // フォームから入力値を取得する
  let name  = document.getElementById("input-name").value;
  let place = document.getElementById("input-place").value;
  let date  = document.getElementById("input-date").value;
  let memo  = document.getElementById("input-memo").value;

  // 名前が空っぽだったら警告して止まる
  if (name === "") {
    alert("名前を入力してください！");
    return;
  }

  // カードを追加する
  let container = document.getElementById("zukan-container");
  container.innerHTML += `
    <div class="card" onclick="alert('${name}だよ！')">
      <h2>${name}</h2>
      <img src="https://placehold.jp/240x180.png?text=${name}"
           alt="${name}" width="200">
      <p>${memo}</p>
      <ul>
        <li>📍 場所：${place}</li>
        <li>📅 日付：${date}</li>
      </ul>
    </div>
  `;

  // フォームをリセット
  document.getElementById("input-name").value  = "";
  document.getElementById("input-place").value = "";
  document.getElementById("input-date").value  = "";
  document.getElementById("input-memo").value  = "";
}

今日のポイント 💡

  • <input type="text"> → 文字を入力できるテキストボックス
  • <button onclick="関数名()"> → クリックで関数を呼び出すボタン
  • function 名前() { ... } → 「命令のまとまり」が関数。呼ばれた時に実行される
  • 要素.value → 入力フォームに書かれた値を取り出す
  • if (名前 === "") { ... } → 「もし〇〇ならば」という条件分岐

チャレンジ問題 🌟

カードを追加したとき、alert('追加したよ！🎉') というメッセージを出してみよう！
