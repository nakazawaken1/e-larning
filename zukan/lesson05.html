第5回：図鑑っぽくタイル状に並べよう！

今日のゴール 🎯

生き物のデータを横に並べて、本物の「図鑑」みたいなタイル表示にする！

学ぶこと 📚

  • 複数の要素をグループにまとめる <div> タグ
  • タグに名前をつける class 属性
  • 要素を横に並べるCSS：display: flex

やってみよう！ 💻

1divタグで生き物をグループにしよう

生き物の情報を <div class="card"> で、全体を <div class="zukan-container"> で囲もう：

<div class="zukan-container">

  <div class="card">
    <h2>アサガオ</h2>
    <img src="..." alt="アサガオの写真" width="200">
    <p>夏にきれいに咲く花です。</p>
    <ul>
      <li>場所：学校の花壇</li>
      <li>日付：7月20日</li>
    </ul>
  </div>

  <div class="card">
    <h2>カブトムシ</h2>
    <img src="..." alt="カブトムシの写真" width="200">
    <p>かっこいいツノがある夏の虫です。</p>
    <ul>
      <li>場所：近くの森</li>
      <li>日付：8月5日</li>
    </ul>
  </div>

</div>

2CSSで横に並べよう（Flexbox）

<style>
  .zukan-container {
    display: flex;    /* 横に並べる魔法！ */
    flex-wrap: wrap;  /* 画面に入らなければ次の行へ */
    gap: 20px;        /* カードの間のすき間 */
  }

  .card {
    background-color: white;
    width: 250px;
    padding: 15px;
  }
</style>

3ブラウザで確認しよう

ブラウザを更新して、生き物がタイル状に横並びになったか確認しよう！

今日のポイント 💡

  • <div> → 複数のタグをひとまとめにする「箱」タグ
  • class="名前" → CSSでデザインを当てるために箱に名前をつける
  • CSSでクラスを使うときは名前の前に .（ドット）をつける（例：.card）
  • display: flex; → 中の箱を横に並べる便利なCSS

チャレンジ問題 🌟

生き物をもう1つ <div class="card"> で追加してみよう！3つが横に並ぶかな？
