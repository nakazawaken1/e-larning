# 第4回：草花や虫の写真を入れよう！

## 今日のゴール 🎯
図鑑にアサガオやカブトムシなど、生き物の写真を追加して本物の図鑑に近づける！

## 学ぶこと 📚
- 画像を表示する `<img>` タグ
- 画像の大きさを変える方法

## やってみよう！ 💻

### ステップ1：画像を準備しよう
1. `zukan` フォルダの中に、`images` という新しいフォルダを作ります。
2. そのの中に、草花や虫の画像を用意して入れましょう。
   （例：`asagao.jpg`, `kabutomushi.jpg` など。用意できない場合は、あとで練習用の画像を使います）

### ステップ2：HTMLに画像を追加しよう
`index.html` の `<h2>` タグの下に、画像を表示する `<img>` タグを追加します。

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>わたしの図鑑</title>
  <!-- 中略 -->
</head>
<body>
  <h1>わたしの図鑑</h1>
  
  <h2>アサガオ</h2>
  <img src="images/asagao.jpg" alt="アサガオの写真" width="200">
  <p>夏にきれいに咲く花です。</p>
  <ul>
    <li>見つけた場所：学校の花壇</li>
    <li>見つけた日：7月20日</li>
  </ul>

  <h2>カブトムシ</h2>
  <img src="images/kabutomushi.jpg" alt="カブトムシの写真" width="200">
  <p>かっこいいツノがある夏の虫です。</p>
  <ul>
    <li>見つけた場所：近くの森</li>
    <li>見つけた日：8月5日</li>
  </ul>
</body>
</html>
```

※メモ：もし手元に画像がない場合は、練習用にインターネット上の仮りの画像を使ってもOK！
例：`<img src="https://placehold.jp/240x180.png?text=アサガオ" alt="写真の代わり" width="200">`

### ステップ3：ブラウザで確認しよう
ブラウザを更新して、図鑑に写真が表示されたか確認しよう！

## 今日のポイント 💡
- `<img>` タグには `</img >` のような**終わりのタグがありません**。
- `src="画像の場所"` → どこに画像があるか（ソース）を指定します。
- `alt="画像の説明"` → 画像が表示されない時に代わりに出る文字です。目の不自由な人や検索エンジンにも役立ちます。
- `width="200"` → 画像の横幅を200ピクセルに設定します。
