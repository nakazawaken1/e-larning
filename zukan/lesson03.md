# 第3回：図鑑に色をつけよう！

## 今日のゴール 🎯
文字に色をつけたり、背景を好きな色にして、図鑑をカラフルにする！

## 学ぶこと 📚
- CSS（スタイルシート）って何？
- 文字の色を変える方法（color）
- 背景の色を変える方法（background-color）

## やってみよう！ 💻

### ステップ1：スタイル（CSS）を書く準備をしよう
HTMLの `<head>` の中に `<style>` タグを追加します。

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>わたしの図鑑</title>
  <style>
    /* ここにCSSを書きます */
  </style>
</head>
<!-- 続きのbodyはそのまま -->
```

### ステップ2：色をつけてみよう
`<style>` の中に、文字の色や背景の色を変えるコードを書いてみよう：

```html
<style>
  body {
    background-color: #f0f8ff; /* うすい水色の背景 */
  }
  
  h1 {
    color: #2e8b57; /* 濃い緑色の文字 */
  }
  
  h2 {
    color: #ff6347; /* トマトみたいな赤色の文字 */
  }
</style>
```

### ステップ3：ブラウザで確認しよう
ブラウザを更新して、図鑑がカラフルになったか確認しよう！

## 今日のポイント 💡
- **CSS（シーエスエス）** はWebページのデザインをするための言葉です。
- `<style>` タグの中には、どの部分（セレクタ）をどうするかを書きます。
- 例：`h1 { color: red; }` は「`<h1>` の色（color）を赤（red）にする」という意味です。
- `#f0f8ff` ような書き方（カラーコード）で、たくさんの色を作ることができます。

## チャレンジ問題 🌟
自分の好きな色を探して、`<h1>` や `<body>` の色を変えてみよう！
（「HTML カラーコード」でインターネット検索すると好きな色のコードが見つかるよ！）
