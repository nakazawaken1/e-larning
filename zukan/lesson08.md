# 第8回：図鑑の写真をクリックして動かそう！

## 今日のゴール 🎯
図鑑の写真をクリックしたときに、画面にメッセージ（アラート）を出す！

## 学ぶこと 📚
- クリックした時の命令（イベント）
- 画面にメッセージを出す `alert()`

## やってみよう！ 💻

### ステップ1：HTMLに「クリック」の目印をつけよう
画像の `<img>` タグに、クリックしたときに動くJavaScriptの命令 `onclick` を追加します。

```html
    <!-- アサガオのカードの中 -->
    <div class="card">
      <h2>アサガオ</h2>
      <!-- ★追加：onclick="alert('アサガオだよ！')" -->
      <img src="https://placehold.jp/240x180.png?text=アサガオ" alt="アサガオの写真" width="200" onclick="alert('アサガオだよ！')">
      <!-- ... -->
    </div>

    <!-- カブトムシのカードの中 -->
    <div class="card">
      <h2>カブトムシ</h2>
      <!-- ★追加：onclick="alert('カブトムシだよ！')" -->
      <img src="https://placehold.jp/240x180.png?text=カブトムシ" alt="カブトムシの写真" width="200" onclick="alert('カブトムシだよ！')">
      <!-- ... -->
    </div>
```

### ステップ2：マウスカーソルの形を変えよう
クリックできることが分かるように、写真の上にマウスが乗ったら「指のマーク（ポインター）」になるようにCSSを追加します。

```html
<style>
  /* ...他の設定... */

  img {
    border-radius: 8px;
    cursor: pointer; /* ←これを追加！マウスが指の形になる */
  }
</style>
```

### ステップ3：ブラウザで確認しよう
1. ブラウザを更新します。
2. 写真の上にマウスをのせると、指のマークに変わりますか？
3. 写真をクリックすると、「〇〇だよ！」というメッセージ（アラート）が出ますか？

## 今日のポイント 💡
- `onclick="..."` → 「もしクリックされたら、...の命令を実行してね」という合図（イベント）です。
- `alert('メッセージ')` → ブラウザの画面にお知らせの小さなポップアップ画面（アラート）を出します。
- メッセージは `'`（シングルクォーテーション）または `"`（ダブルクォーテーション）で囲みます。
- `cursor: pointer;` → マウスが上にのった時に、よくリンクなどに使われる指の形に変えるCSSです。

## チャレンジ問題 🌟
アラートのメッセージをもっと面白くしてみよう！
例：`onclick="alert('み〜つけた！')"`
