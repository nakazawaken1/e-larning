第8回：図鑑の写真をクリックして動かそう！

今日のゴール 🎯

図鑑の写真をクリックしたときに「ポップアップ（アラート）」でメッセージを出す！

学ぶこと 📚

  • クリックした時の命令（イベント）：onclick
  • 画面にメッセージを出す：alert()
  • マウスカーソルを指マークにする：cursor: pointer

やってみよう！ 💻

1imgタグに onclick を追加しよう

<div class="card">
  <h2>アサガオ</h2>
  <img
    src="images/asagao.jpg"
    alt="アサガオの写真"
    width="200"
    onclick="alert('アサガオだよ！')"
  >
  ...
</div>

2カーソルを指の形にしよう

img {
  border-radius: 8px;
  cursor: pointer;  /* ← これを追加！ */
}

3実際にクリックしてみよう（練習）

↓ この画像をクリックしてみよう！

今日のポイント 💡

  • onclick="..." → 「クリックされたら...を実行してね」という合図（イベント属性）
  • alert('メッセージ') → 画面にポップアップのお知らせを出す
  • メッセージは '（シングルクォーテーション）または "（ダブルクォーテーション）で囲む
  • cursor: pointer; → マウスを乗せると指マークに変わる

チャレンジ問題 🌟

アラートのメッセージをもっと面白くしてみよう！
例：onclick="alert('み〜つけた！🐛')"
