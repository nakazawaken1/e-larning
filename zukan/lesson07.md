# 第7回：プログラミングの準備！JavaScriptって？

## 今日のゴール 🎯
Webページを動かすための言葉「JavaScript」の基本を知る！

## 学ぶこと 📚
- JavaScript（ジャバスクリプト）って何？
- `<script>` タグの使い方
- デベロッパーツール（コンソール）の使い方
- コンピュータに言葉を覚えさせる「変数（へんすう）」

## やってみよう！ 💻

### ステップ1：JavaScriptを書く準備
HTMLの中に、JavaScriptを書くための `<script>` タグを追加します。
`</body>` の直前に書くのが一般的です。

```html
<!-- 前回のコードの続き -->
  </div> <!-- zukan-containerの終わり -->

  <!-- ここからJavaScriptを書く準備！ -->
  <script>
    // ここにJavaScriptを書きます
    console.log("図鑑のプログラムがスタートしました！");
  </script>
</body>
</html>
```

### ステップ2：コンソールで確認しよう
`console.log()` は、ブラウザの「裏側」にこっそりメッセージを表示する魔法の言葉です。

1. ブラウザを更新します。（画面には何も変わりません）
2. キーボードで、デベロッパーツールを開きます。
   - Windowsの場合：`F12` キー または `Ctrl + Shift + I`
   - Macの場合：`Command + Option + I`
3. 「Console（コンソール）」というタブ（メニュー）を開きます。
4. 「図鑑のプログラムがスタートしました！」という文字が出たら成功！

### ステップ3：言葉を覚えさせよう（変数）
一時的にデータを覚えておく「箱」を変数（へんすう）と呼びます。
`let` という言葉を使って箱を作ります。`<script>` の中に追加してみましょう。

```javascript
  <script>
    // 挨拶を覚える箱（変数）を作る
    let aisatsu = "こんにちは！";

    // 箱の中身をコンソールに表示する
    console.log(aisatsu);

    // 生き物の名前を入れる箱を作る
    let mushi = "カブトムシ";
    console.log("私が見つけたのは " + mushi + " です。");
  </script>
```

ブラウザを更新して、コンソールにメッセージが表示されるか確認しよう！

## 今日のポイント 💡
- **JavaScript（JS）** は、Webページに「動き」や「考える力」をつけるプログラミング言語です。
- `<script>` の中に書きます。
- `console.log()` → 開発者用の裏画面（コンソール）にメッセージを出します。（プログラマーの大事な道具！）
- `let 名前 = "中身";` → データを入れておく箱（変数）を作ります。

## チャレンジ問題 🌟
