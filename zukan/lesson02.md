# 第2回：図鑑の説明を書こう！

## 今日のゴール 🎯
図鑑に草花や虫の名前や、見つけた場所、メモを追加する！

## 学ぶこと 📚
- 文章を表示する `<p>` タグ
- 箇条書きを作る `<ul>` と `<li>` タグ

## やってみよう！ 💻

### ステップ1：HTMLを追加しよう
前回作ったファイル（例：`index.html`）の `<body>` の中に、以下のコードを追加してみよう：

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>わたしの図鑑</title>
</head>
<body>
  <h1>わたしの図鑑</h1>
  
  <h2>アサガオ</h2>
  <p>夏にきれいに咲く花です。</p>
  <ul>
    <li>見つけた場所：学校の花壇</li>
    <li>見つけた日：7月20日</li>
  </ul>

  <h2>カブトムシ</h2>
  <p>かっこいいツノがある夏の虫です。</p>
  <ul>
    <li>見つけた場所：近くの森</li>
    <li>見つけた日：8月5日</li>
  </ul>
</body>
</html>
```

### ステップ2：ブラウザで確認しよう
1. ファイルを上書き保存する
2. ブラウザの「更新（リロード）」ボタンを押して、文字が追加されたか確認しよう！

## 今日のポイント 💡
- `<h2>` → `<h1>` より少し小さな見出し（生き物の名前にぴったり！）
- `<p>` → 普通の文章（段落）を書くタグ
- `<ul>` と `<li>` → セットで使って「・（黒丸）」のリスト（箇条書き）を作る
