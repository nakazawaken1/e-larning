第1回：図鑑のベースを作ろう！
最初のHTMLファイルを作ろう

今日のゴール 🎯

「わたしの図鑑」というタイトルが表示されるWebページを作る！

学ぶこと 📚

  • Webページはどうやって動いているの？
  • HTMLファイルって何？
  • ブラウザでファイルを開く方法

やってみよう！ 💻

1ファイルを作ろう

  1. パソコンで「メモ帳」（Windowsの場合）または「テキストエディット」（Macの場合）を開く
  2. 新しいファイルを作って index.html という名前で保存する

2HTMLを書こう

以下のコードを書いてみよう：

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>わたしの図鑑</title>
</head>
<body>
  <h1>わたしの図鑑</h1>
</body>
</html>

3ブラウザで開こう

  1. 保存した index.html ファイルをダブルクリック
  2. ブラウザに「わたしの図鑑」と表示されたら成功！

今日のポイント 💡

  • <!DOCTYPE html> → 「これはHTMLファイルです」とブラウザに教える
  • <html> → HTML全体を囲む
  • <head> → ページの設定を書く場所
  • <body> → 実際に表示される内容を書く場所
  • <h1> → 一番大きな見出し

チャレンジ問題 🌟

<h1>わたしの図鑑</h1> のテキストを、自分だけのオリジナルタイトルに変えてみよう！

<h1>けんいちの昆虫図鑑</h1>
