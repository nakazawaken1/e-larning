第10回：データから
図鑑のタイルを作ろう！

今日のゴール 🎯

JavaScriptのデータを使って、タイルカードを自動生成する！HTMLに手書きしなくてよくなる！

学ぶこと 📚

  • idでHTML要素を取得する：document.getElementById()
  • 配列を繰り返し処理する：forEach()
  • JavaScriptでHTMLを作って追加する：innerHTML
  • テンプレートリテラル：`文字列${変数}`

やってみよう！ 💻

1HTMLのカードを消して入れ物だけ残す

手書きの <div class="card"> をすべて消し、空の入れ物だけにします：

<body>
  <h1>わたしの図鑑</h1>

  <!-- この中はJavaScriptが自動で作る！ -->
  <div id="zukan-container" class="zukan-container">
  </div>
</body>

2JavaScriptでカードを自動生成する

let zukanData = [
  { name: "アサガオ",
    image: "https://placehold.jp/240x180.png?text=アサガオ",
    description: "夏にきれいに咲く花です。",
    place: "学校の花壇", date: "7月20日" },
  { name: "カブトムシ",
    image: "https://placehold.jp/240x180.png?text=カブトムシ",
    description: "かっこいいツノがある夏の虫です。",
    place: "近くの森", date: "8月5日" }
];

// idでHTML要素を取得
let container = document.getElementById("zukan-container");

// 配列を1件ずつ処理して card を作る
zukanData.forEach(function(ikimono) {
  container.innerHTML += `
    <div class="card" onclick="alert('${ikimono.name}だよ！')">
      <h2>${ikimono.name}</h2>
      <img src="${ikimono.image}" alt="${ikimono.name}" width="200">
      <p>${ikimono.description}</p>
      <ul>
        <li>📍 場所：${ikimono.place}</li>
        <li>📅 日付：${ikimono.date}</li>
      </ul>
    </div>
  `;
});

今日のポイント 💡

  • document.getElementById("名前") → id="名前" がついた要素を取得する
  • 配列.forEach(function(item){ ... }) → 配列の中のデータを1つずつ取り出して処理する
  • バッククォート ` `（テンプレートリテラル）で ${変数} を埋め込める
  • innerHTML → HTML要素の中身を読み書きできる

チャレンジ問題 🌟

zukanData の配列に3つ目の生き物を追加してみよう！HTMLは変えなくても、タイルが自動で増えるよ！
