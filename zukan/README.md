# 図鑑Webサービス教材

草花や虫の写真・場所・メモなどを登録して、タイル表示できる「図鑑Webサービス」を作る全12回の入門プログラミング教材です。

## 学習コース

| 回 | テーマ | HTML完成形 |
|---|---|---|
| 第1回 | 図鑑のベースを作ろう！ | [lesson01.html](lesson01.html) |
| 第2回 | 図鑑の説明を書こう！ | [lesson02.html](lesson02.html) |
| 第3回 | 図鑑に色をつけよう！ | [lesson03.html](lesson03.html) |
| 第4回 | 草花や虫の写真を入れよう！ | [lesson04.html](lesson04.html) |
| 第5回 | 図鑑っぽくタイル状に並べよう！ | [lesson05.html](lesson05.html) |
| 第6回 | 図鑑のデザインをかっこよくしよう！ | [lesson06.html](lesson06.html) |
| 第7回 | プログラミングの準備！JavaScriptって？ | [lesson07.html](lesson07.html) |
| 第8回 | 図鑑の写真をクリックして動かそう！ | [lesson08.html](lesson08.html) |
| 第9回 | 図鑑のデータをJavaScriptで持とう！ | [lesson09.html](lesson09.html) |

## 始め方

1. `index.html` をブラウザで開くと、全レッスン一覧が表示されます。
2. 各レッスンの「📄 テキスト」を読んでからコードを書き、「🌐 HTML」で完成形を確認しましょう。

## 対象

- プログラミング初心者（小中学生〜大人まで）
- HTML / CSS / JavaScript を順番に学びたい方
