第6回：図鑑のデザインをかっこよくしよう！

今日のゴール 🎯

カードの角を丸くしたり、影をつけて、本物のアプリみたいにかっこよくする！

学ぶこと 📚

  • 角を丸くする：border-radius
  • 影をつける：box-shadow
  • テキストを真ん中にする：text-align: center
  • リストの「・」を消す：list-style: none

やってみよう！ 💻

1カードをおしゃれにしよう

.card の CSS に追加しよう：

.card {
  background-color: white;
  width: 250px;
  padding: 15px;
  border-radius: 10px;          /* 角を丸くする */
  box-shadow: 0 4px 8px         /* ふんわりした影 */
              rgba(0,0,0,0.1);
  text-align: center;           /* 文字を真ん中に */
}

2リストとimg もおしゃれにしよう

ul {
  list-style: none;  /* 「・」を消す */
  padding: 0;
  color: #666;
  font-size: 14px;
  text-align: left;
}

img {
  border-radius: 8px;  /* 写真の角も丸くする */
}

3完成イメージ（ブラウザで確認しよう）

アサガオ

夏にきれいに咲く花です。

今日のポイント 💡

  • border-radius: 10px; → 角の丸みをつける。数字が大きいほど丸くなる
  • box-shadow: 0 4px 8px rgba(0,0,0,0.1); → やわらかい影をつける
  • text-align: center; → 文字・画像を真ん中にそろえる
  • list-style: none; → 「・」などのリストの記号を消す

チャレンジ問題 🌟

border-radius: 10px; の数字を 30px や 50px に変えてみよう！角はどんな形になったかな？
