第9回：図鑑のデータを
JavaScriptで持とう！

今日のゴール 🎯

図鑑の情報を「配列（はいれつ）」と「オブジェクト」を使ってJavaScriptでまとめる！

学ぶこと 📚

  • 1つの生き物の情報をまとめる「オブジェクト」 { }
  • 複数のデータのリスト「配列（はいれつ）」 [ ]
  • データとデザインを「分けて考える」考え方

やってみよう！ 💻

1オブジェクトを作ろう

1つの生き物のデータをオブジェクト（{ }）にまとめよう：

let asagao = {
  name:        "アサガオ",
  image:       "images/asagao.jpg",
  description: "夏にきれいに咲く花です。",
  place:       "学校の花壇",
  date:        "7月20日"
};

// プロパティの中身を取り出す
console.log(asagao.name);  //「アサガオ」と表示！

2配列に複数のデータをまとめよう

let zukanData = [
  {
    name: "アサガオ",
    image: "https://placehold.jp/240x180.png?text=アサガオ",
    description: "夏にきれいに咲く花です。",
    place: "学校の花壇",
    date: "7月20日"
  },
  {
    name: "カブトムシ",
    image: "https://placehold.jp/240x180.png?text=カブトムシ",
    description: "かっこいいツノがある夏の虫です。",
    place: "近くの森",
    date: "8月5日"
  }
];

console.log(zukanData);  // コンソールで中身を確認！

3コンソールで確認しよう

  1. ブラウザでファイルを更新する（表示は変わらない）
  2. デベロッパーツールのコンソールを開く
  3. Array(2) のような表示が出たらデータが入っている！▶ マークをクリックして中身を確認しよう

今日のポイント 💡

  • オブジェクト { name: "アサガオ", place: "花壇" } → 1つのものについて複数のプロパティ（名前と値）をセットにしたデータ
  • 配列（はいれつ） [ データ1, データ2 ] → 複数のデータを順番に並べたリスト
  • オブジェクトの中身は オブジェクト名.プロパティ名 で取り出せる
  • アプリ開発では「データ（JS）」と「見た目（HTML）」を分けて考えることが大切！

チャレンジ問題 🌟

zukanData の配列の中に、好きな生き物のオブジェクトを もう1つ 追加してみよう！
