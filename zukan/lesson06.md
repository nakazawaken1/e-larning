# 第6回：図鑑のデザインをかっこよくしよう！

## 今日のゴール 🎯
カードの角を丸くしたり、影をつけたりして、本物のアプリみたいにかっこよくする！

## 学ぶこと 📚
- 角を丸くするCSS `border-radius`
- 影をつけるCSS `box-shadow`
- テキストの配置を変える `text-align`

## やってみよう！ 💻

### ステップ1：カードをもっとおしゃれにしよう
`<style>` の中の `.card` に、デザインを良くするCSSを追加します。

```html
<style>
  /* 前回の部分は省略してます。そのまま残してね！ */
  
  .card {
    background-color: white;
    width: 250px;
    padding: 15px;
    
    /* ここから追加！ */
    border-radius: 10px; /* 角を少し丸くする */
    box-shadow: 0 4px 8px rgba(0,0,0,0.1); /* ふんわりとした影をつける */
    text-align: center; /* 文字や画像を真ん中にそろえる */
  }
</style>
```

### ステップ2：リストの「・」を消してスッキリさせよう
`<ul>` のリストの点が不要な場合は、CSSで消すことができます。
`<style>` に `ul` の設定を追加しましょう。

```html
<style>
  /* ...他の設定... */

  ul {
    list-style: none; /* リストの「・」を消す */
    padding: 0; /* 左側のスキマをなくす */
    color: #666; /* 文字を少しうすいグレーにする */
    font-size: 14px; /* 文字を少し小さくする */
  }
</style>
```

※ついでに画像の角も丸くするともっとかわいくなりますよ！
```html
<style>
  img {
    border-radius: 8px; /* 写真の角も丸くする */
  }
</style>
```

### ステップ3：ブラウザで確認しよう
ブラウザを更新して、デザインがかっこよくなったか確認しよう！

## 今日のポイント 💡
- `border-radius` → 箱の角や写真の角を丸くします。数字が大きいほど丸くなります。
- `box-shadow` → 箱に影をつけます。アプリのような立体感が出ます！
- `text-align: center;` → 文字（テキスト）を真ん中（センター）にそろえます。
- `list-style: none;` → `<ul>` や `<li>` の記号を消します。
