最初のHTMLファイルを作ろう

今日のゴール 🎯

「時間割」というタイトルが表示されるWebページを作る！

学ぶこと 📚

  • Webページはどうやって動いているの？
  • HTMLファイルって何？
  • ブラウザでファイルを開く方法

HTMLファイルの構造イメージ

やってみよう！ 💻

1ファイルを作ろう

  1. パソコンで「メモ帳」（Windowsの場合）または「テキストエディット」（Macの場合）を開く
  2. 新しいファイルを作って index.html という名前で保存する

2HTMLを書こう

以下のコードを書いてみよう：

<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>わたしの時間割</title>
</head>
<body>
  <h1>時間割</h1>
</body>
</html>

3ブラウザで開こう

  1. 保存した index.html ファイルをダブルクリック
  2. ブラウザに「時間割」と表示されたら成功！

今日のポイント 💡

  • <!DOCTYPE html> → 「これはHTMLファイルです」とブラウザに教える
  • <html> → HTML全体を囲む
  • <head> → ページの設定を書く場所
  • <body> → 実際に表示される内容を書く場所
  • <h1> → 一番大きな見出し

チャレンジ問題 🌟

<h1>時間割</h1> の下に、自分の名前を <p> タグで追加してみよう！

例：

<p>○○小学校 ○年○組</p>
