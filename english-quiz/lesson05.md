# 第5回：クリックで動かそう（JavaScript）

## 今日のゴール 🎯
ボタンを押した時に「正解！」や「ハズレ！」と画面に表示させる！

## 学ぶこと 📚
- JavaScriptってなに？（動きをつける言葉）
- 関数（`function`）を作ってみよう
- ボタンが押されたことを知る（`onclick`）
- メッセージを出す（`alert`）

## やってみよう！ 💻

### ステップ1：プログラムを書く場所を作ろう
`</body>` の直前に `<script>` タグを追加します。

```html
  <script>
    function checkAnswer() {
      alert("ボタンが押されたよ！");
    }
  </script>
</body>
```

### ステップ2：ボタンとつなげよう
HTMLのボタンに `onclick="checkAnswer()"` を追加します。

```html
<button onclick="checkAnswer()">りんご</button>
```

これでボタンを押すとメッセージが出るようになります。

4. ボタンがクリックされたことをプログラムに伝える属性は何ですか？
5. メッセージを表示するためのJavaScriptの命令は何ですか？
