# 第11回：タブレットでも見やすくしよう

## 今日のゴール 🎯
スマホやタブレット、パソコン、どんな画面でもきれいに見えるようにする！

## 学ぶこと 📚
- 「レスポンシブ」という考え方
- 画面の幅に合わせる（`%` や `vw`）
- メディアクエリ（`@media`）の基本

## やってみよう！ 💻

### ステップ1：幅を柔らかくしよう
`#quiz-container` の幅を固定ではなく `%` にしてみます。

```css
#quiz-container {
  width: 90%;          /* 画面の90%の幅にする */
  max-width: 500px;   /* でも500pxよりは大きくしない */
}
```

### ステップ2：大きな画面の時だけ変える
```css
@media (min-width: 600px) {
  h1 {
    font-size: 60px; /* パソコンのときは文字を大きく！ */
  }
}
```

## 理解度チェック ✅
1. 画面の大きさに合わせてデザインが自動で変わることを何と言いますか？
2. 画面の横幅（パソコンやスマホの幅）に合わせてサイズを決める単位は何ですか？ (ヒント: `%` や `??`)
3. 特定の画面サイズ（スマホなど）のときだけ別のデザインにするための命令は何ですか？
4. スマホ対応のためにHTMLの `<head>` に入れる「画面の設定（おまじない）」を何と言いますか？
5. `max-width: 500px` と指定した場合、画面が 800px あったとしても、箱の幅は最大で何pxになりますか？
