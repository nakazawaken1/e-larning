# 第7回：100個の単語リストを作ろう

## 今日のゴール 🎯
たくさんの英単語を「リスト」にして、プログラムで使いやすくする！

## 学ぶこと 📚
- たくさんのデータをまとめる「配列」（`[]`）
- 意味と名前をセットにする「オブジェクト」（`{}`）
- 100個の単語データを作ってみる

## やってみよう！ 💻

### ステップ1：単語のリストを作ろう
```javascript
const wordList = [
  { english: "apple", japanese: "りんご" },
  { english: "banana", japanese: "バナナ" },
  { english: "cat", japanese: "ねこ" },
  { english: "dog", japanese: "いぬ" },
  // ...ここから100個繋げていくよ！
];
```

### ステップ2：100個のリストの例
教材の最後に、100個のリストを載せておくので、コピーして使ってみよう！

## 理解度チェック ✅
1. たくさんのデータを順番に並べてまとめるものを何と言いますか？
2. 配列を作るときに使うカッコの記号は何ですか？
3. 「名前」と「値」をセットにしてデータをまとめるものを何と言いますか？
4. オブジェクトを作るときに使うカッコの記号は何ですか？
5. 値が変わらない（定数）箱を作るときに使うキーワードは何ですか？
