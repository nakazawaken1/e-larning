# 第6回：正解・不正解を判定しよう

## 今日のゴール 🎯
押したボタンが正しいかどうかをコンピュータに判断させる！

## 学ぶこと 📚
- コンピュータに値を渡す（引数）
- 正解かどうか比べる（`if`文）
- 「＝」の使い方（JavaScriptでは `===` を使うよ）

## やってみよう！ 💻

### ステップ1：正解を教えてあげよう
ボタンごとに違う名前（文字）を渡すようにします。

```html
<button onclick="checkAnswer('りんご')">りんご</button>
<button onclick="checkAnswer('みかん')">みかん</button>
```

### ステップ2：もし〜なら（if）を使おう
```javascript
function checkAnswer(choice) {
  if (choice === 'りんご') {
    alert("正解！大当たり！🎉");
  } else {
    alert("ざんねん、ハズレだよ...😢");
  }
}
```

## 理解度チェック ✅
1. 「もし〜なら」という判断をさせるためのキーワードは何ですか？
2. 「そうじゃなかったら」という、反対のケースを書くときに使うキーワードは何ですか？
3. JavaScriptで、右と左が「同じである」ことを比べる記号は何ですか？
4. 関数を呼ぶときに、カッコの中に書く「コンピュータに渡す値」を何と言いますか？
5. `if (choice === 'りんご')` というプログラムで、 `choice` の中身が 'バナナ' だった場合、ifの中の命令は実行されますか？

## 今日のポイント 💡
- `if ( ... ) { ... }` は「もし〜だったら」という条件分岐。
- `===` は右と左が「同じ」という意味。
- `else` は「そうじゃなかったら」という意味だよ。
