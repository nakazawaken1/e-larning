# 第1回：Webページを作ってみよう！最初のHTMLファイル

## 今日のゴール 🎯
「英単語クイズ」というタイトルが表示されるWebページを作る！

## 学ぶこと 📚
- Webページはどうやって動いているの？
- HTMLファイルって何？
- ブラウザでファイルを開く方法

## やってみよう！ 💻

### ステップ1：ファイルを作ろう
1. パソコンで「メモ帳」（Windowsの場合）または「テキストエディット」（Macの場合）を開く
2. 新しいファイルを作って `index.html` という名前で保存する

### ステップ2：HTMLを書こう
以下のコードを書いてみよう：

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>英単語クイズ</title>
</head>
<body>
  <h1>英単語クイズ</h1>
  <p>これからクイズアプリを作っていくよ！</p>
</body>
</html>
```

### ステップ3：ブラウザで開こう
1. 保存した `index.html` ファイルをダブルクリック
2. ブラウザに「英単語クイズ」と表示されたら成功！

## 今日のポイント 💡
- `<!DOCTYPE html>` → 「これはHTMLファイルです」とブラウザに教える
- `<html>` → HTML全体を囲む
- `<head>` → ページの設定を書く場所
- `<body>` → 実際に表示される内容を書く場所
- `<h1>` → 一番大きな見出し
- `<p>` → 段落（文章）を書く場所

## チャレンジ問題 🌟
