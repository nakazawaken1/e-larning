# 第10回：アニメーションで楽しくしよう

## 今日のゴール 🎯
正解した時に画面をキラキラさせたり、ボタンを揺らしたりして楽しくする！

## 学ぶこと 📚
- CSSでアニメーションを作る（`@keyframes`）
- アニメーションを動かす時間（`duration`）
- JavaScriptでCSSを切り替える

## やってみよう！ 💻

### ステップ1：揺れる動きを作ろう
```css
@keyframes shake {
  0% { transform: translateX(0); }
  25% { transform: translateX(-5px); }
  50% { transform: translateX(5px); }
  100% { transform: translateX(0); }
}

.wrong-animation {
  animation: shake 0.3s;
}
```

### ステップ2：正解のときの色
正解したボタンをピカッと光らせてみよう！

## 理解度チェック ✅
1. アニメーションの「動きのステップ」を決めるためのキーワードは何ですか？
2. 作った動きを要素に割り当てるためのプロパティは何ですか？
3. 要素を動かしたり、大きさを変えたりするときに使うプロパティは何ですか？
4. アニメーションの時間を指定するときに使う単位（秒）は何ですか？
5. JavaScriptで、特定のCSSクラス（動きのルール）を後から付け足すための命令は何ですか？
