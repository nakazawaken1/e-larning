# 第2回：色やデザインを変えよう（CSS）

## 今日のゴール 🎯
文字の色や背景色を変えて、タイトルをかっこよくデザインする！

## 学ぶこと 📚
- CSSってなに？（デザインを担当する仕組み）
- `style` タグの使い方
- 色（`color`）と背景（`background-color`）の変え方
- 文字を真ん中に寄せる（`text-align`）方法

## やってみよう！ 💻

### ステップ1：styleタグを追加しよう
`<head>` の中に `<style>` タグを書こう。

```html
<head>
  <meta charset="UTF-8">
  <title>英単語クイズ</title>
  <style>
    /* ここにデザインを書くよ */
    body {
      background-color: #f0f9ff; /* 水色っぽくする */
      text-align: center;       /* 全部真ん中に寄せる */
      font-family: sans-serif;  /* 文字の形をすっきりさせる */
    }
  </style>
</head>
```

### ステップ2：タイトルに色をつけよう
`h1` タグの文字の色を変えてみよう。

```css
h1 {
  color: #0ea5e9; /* きれいな青色 */
  font-size: 40px;
  margin-top: 50px;
}
```

## 今日のポイント 💡
- CSSは「どこに（セレクタ）」「何を（プロパティ）」「どうする（値）」というルールで書くよ！
- `color` → 文字の色
- `background-color` → 背景の色
- `text-align: center` → 真ん中寄せ

## チャレンジ問題 🌟
自分の好きな色を探して、背景色を変えてみよう！
（例：`#ffedd5` はうすいオレンジ、`#fef2f2` はうすいピンクだよ）
