# 第6回：マウスでクリックしたら動かそう

## 今日のゴール 🎯
時間割のマスをクリックすると教科名を入力できるようにする！

## 学ぶこと 📚
- マスをクリックした時の処理
- 入力ダイアログの使い方
- HTMLの内容を書き換える方法

## やってみよう！ 💻

### ステップ1：マスにクリック機能を追加しよう
`<script>` の中を以下のように書き換え：

```javascript
// すべての td 要素を取得
const cells = document.querySelectorAll('td');

// それぞれのマスにクリックイベントを追加
cells.forEach(function(cell) {
  cell.onclick = function() {
    // 入力ダイアログを表示
    const subject = prompt('教科名を入力してね');
    
    // 入力された内容をマスに表示
    if (subject !== null) {
      cell.textContent = subject;
    }
  };
});
```

### ステップ2：試してみよう
1. ブラウザを更新
2. 時間割の空いているマスをクリック
3. 教科名（例：「国語」）を入力
4. OKボタンを押すとマスに表示される！

## 今日のポイント 💡
- `querySelectorAll` → 条件に合う要素を全部取得
- `forEach` → 取得した要素を1つずつ処理
- `prompt` → 入力ダイアログを表示
- `textContent` → 要素のテキスト内容

## プロンプトの戻り値について 📝
- OKを押した → 入力した文字が返る
- キャンセルを押した → `null` が返る
- だから `if (subject !== null)` でチェックが必要！
