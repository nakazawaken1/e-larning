# 第5回：JavaScriptって何？最初のプログラム

## 今日のゴール 🎯
ボタンを押すとメッセージが表示されるページを作る！

## 学ぶこと 📚
- JavaScriptでできること
- メッセージを表示する方法
- ボタンを追加する方法

## やってみよう！ 💻

### ステップ1：ボタンを追加しよう
`</table>` の下に以下を追加：

```html
<button id="helloButton">あいさつする</button>
```

### ステップ2：JavaScriptを書く場所を作ろう
`</body>` の直前に `<script>` タグを追加：

```html
  <script>
    // ここにJavaScriptを書くよ
  </script>
</body>
```

### ステップ3：ボタンをクリックしたら動くようにしよう

```html
<script>
  // ボタンを取得
  const button = document.getElementById('helloButton');
  
  // クリックしたら実行
  button.onclick = function() {
    alert('こんにちは！時間割アプリへようこそ！');
  };
</script>
```

## 今日のポイント 💡
- `document.getElementById` → HTMLの要素を取得する
- `onclick` → クリックした時に動く
- `alert` → ポップアップでメッセージを表示
- `function() { }` → 実行したい処理を書く場所
- `//` → コメント（メモ書き、実行されない）

## 開発者ツールを使ってみよう 🔧
1. ブラウザでF12キーを押す
2. 「Console（コンソール）」タブを開く
3. `console.log('Hello!');` と入力してEnter
4. メッセージが表示される！

## チャレンジ問題 🌟

2. `console.log('時間割アプリ開発中！');` を追加してみよう
3. もう1つボタンを追加して、違うメッセージを表示してみよう
