## やってみよう！ 💻

### ステップ1：CSSを書く場所を作ろう
`<head>` の中に `<style>` タグを追加しよう：

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>わたしの時間割</title>
  <style>
    /* ここにCSSを書くよ */
  </style>
</head>
<body>
  <h1>時間割</h1>
</body>
</html>
```

### ステップ2：色をつけよう
`<style>` の中に以下を書こう：

```css
body {
  background-color: #f0f8ff;
}

h1 {
  color: #4169e1;
  text-align: center;
}
```

### ステップ3：保存してブラウザで確認
ファイルを保存して、ブラウザを更新（F5キー）してみよう！

## 今日のポイント 💡
- `background-color` → 背景の色
- `color` → 文字の色
- `text-align: center` → 中央揃え
- 色は `#` で始まるカラーコードで指定

## よく使う色コード 🎨
| 色 | コード |
|---|---|
| 赤 | #ff0000 |
| 青 | #0000ff |
| 緑 | #00ff00 |
| 黄 | #ffff00 |
| ピンク | #ffc0cb |
| オレンジ | #ffa500 |
